# Fonts Required

Download these fonts and place in this directory:

1. **Press Start 2P** (pixel font for headers)
   - URL: https://fonts.google.com/specimen/Press+Start+2P
   - File: PressStart2P-Regular.ttf
   - Usage: Wave announcements, score popups, game over text

2. **Inter** (clean sans-serif for body/numbers)
   - URL: https://fonts.google.com/specimen/Inter
   - File: Inter-Regular.ttf, Inter-Bold.ttf
   - Usage: HUD numbers, settings, body text

After downloading, import them in Godot and create .tres font resources.
